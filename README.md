# Workout Tracker

A personal Python project built to strengthen practical experience with SQLite, SQL, and backend application logic through workout data tracking.

## Features

- Create workout splits
- Store exercises by workout day
- Start workout sessions
- Log sets, reps, and weight
- Edit or delete logged sets

## What I Practiced

- SQLite database integration
- Relational schema design
- Foreign key relationships
- Session and set tracking workflows
- CLI-based user interaction
- CRUD operations across related tables

## Notes

This project is a work in progress and was primarily built to deepen my understanding of SQL, table design, and connected data models.
